💘 Valentine Proposal Web App

A cute and interactive Valentine proposal web app built using Flask 💗
This project creates a playful experience where the "No" option becomes harder to choose over time 😏

---

✨ Features

- 💞 Interactive Yes/No buttons
- 😈 "Yes" button grows on every "No" click
- 🥺 "No" button shrinks gradually
- 🐱 Cute images that change with each attempt
- 🎉 Final "Yes" page with animation & message

---

🛠 Tech Stack

- Python (Flask)
- HTML, CSS
- Basic JavaScript (for animations)

🚀 How to Run Locally

1. Clone the repository

git clone https://github.com/your-username/your-repo-name.git
cd romantic-web-app

2. Install dependencies

pip install -r requirements.txt

3. Run the app

python app.py

4. Open in browser

http://127.0.0.1:5000/
